# Equity-excel-table
Dataset analyzing workplace equality across Daikibo factories. It tracks equality scores and classifications (Fair, Unfair, Highly Discriminative) for various job roles across multiple global locations, providing insights into organizational fairness and diversity.
This repository contains a dataset analyzing the state of equality within various Daikibo Factory locations (Meiyo, Seiko, and Berlin). The data maps specific job roles to equality metrics, allowing for an audit of organizational fairness across different levels of the corporate hierarchy.Dataset StructureThe dataset is provided in a CSV format with the following columns:ColumnDescriptionFactoryThe specific factory location (e.g., Daikibo Factory Meiyo, Daikibo Berlin).Job RoleThe professional designation, ranging from C-Level and VP to Engineers and Operational Support.Equality ScoreA numerical value representing the calculated equality metric for that specific role and location.Equality classA qualitative classification based on the score: Fair, Unfair, or Highly Discriminative.Key ClassificationsThe equality status is categorized as follows:Fair: Indicates roles where equality metrics meet acceptable standards.Unfair: Indicates a moderate level of disparity that requires attention.Highly Discriminative: Indicates significant inequality gaps that may require immediate intervention or policy changes.Potential Use CasesHR Auditing: Identifying which departments or locations require diversity and inclusion training.Data Visualization: Creating heatmaps of equality across different factory locations.Trend Analysis: Comparing how management roles fare against operational roles in terms of workplace fairness.File InformationFile Name: Task 5 Equality Table.xlsx - Sheet1.csvFormat: CSV (Comma Separated Values)
